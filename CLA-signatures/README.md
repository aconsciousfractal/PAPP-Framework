# CLA Signatures

This directory contains signed Contributor License Agreements from contributors to PAPP.

## For Contributors

Before contributing code to PAPP, you must sign the [Contributor License Agreement](../CLA.md).

### How to Sign

1. Read the [CLA](../CLA.md)
2. Create a file: `CLA-signed-[YourGitHubUsername].md`
3. Fill in your information (see template below)
4. Submit via Pull Request to this directory

### Template

```markdown
# CLA Signature

I have read and agree to the PAPP Contributor License Agreement (CLA v1.0).

**Full Name**: Your Full Name
**GitHub Username**: [@YourUsername](https://github.com/YourUsername)
**Email**: your.email@example.com
**Date**: 2026-MM-DD
**Signature**: Your Full Name

## Contributions
I confirm this CLA applies to all my past and future contributions to PAPP.
```
